# ProGen

A small CLI tool for generating projects.

## Installation
Do a global npm install:
```
npm install -g @patrand/progen
```


## Usage
Just use it like this:
```shell
generate <type>
```
*Possible types can be found at the **Feature** section* 

To list all types just type
```
generate help
```

## Features
- React 
- Node Module
- NestJS
